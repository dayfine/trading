# Weekly pick records

Generated by `generate_weekly_snapshot` (M6.6). Layout:
`dev/weekly-picks/<system-version>/<YYYY-MM-DD>.sexp`.

Produced by running the Weinstein screener cascade on cached bars for a given
as-of date; consumed by `trace_picks` / `diff_picks` / `render_weekly_report`.

## Live arming config

Live runs MUST pass `--config-overrides dev/weekly-picks/live-config-overrides.sexp`
— the explicit arming config for mechanisms accepted for live use (code
defaults stay no-op per experiment-flag R1). Currently armed (2026-07-14):
`extension_stop (2.0, 0.25)` (ledger 2026-07-14-extension-stop-insurance-accept)
and `reject_declining_ma_long_entry` (#1775 ARM-FOR-BROAD). Same shape as a
scenario's `config_overrides`; unknown keys fail loudly via `Overlay_validator`.
